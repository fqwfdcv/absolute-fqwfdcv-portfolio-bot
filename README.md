## 🚀 Быстрый старт

```bash
git clone https://github.com/absolute-fqwfdcv/portfolio-telegram-bot.git
cd portfolio-telegram-bot
npm install
```

Создай `.env` и добавь токен:

```
TOKEN=твой_токен_бота
```

Запусти:

```bash
node bot.js
```
